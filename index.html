WP One Metric

WP One MetricはOne Content Metric to Rule Them Allの指標に基づき、コンテンツの評価を可視化する事ができるWordPressプラグインです。 解析は過去40記事分について実施されます。その後は1日に1回解析が走って自動更新されていく仕組みです。

解析自体は以下の数値を指標として実施しています。

  • Google Analyticsのユニークページビュー
  • facebookの拡散数
  • Twitterでの拡散数

改善提案はGitHub にプルリクエストを送ってください。

How use

  1. Set up on Settings > WP One Metric Setting
  2. Push Analyze button on Dashboard WP One Metric

  2. Activate the plugin through the 'Plugins' menu in WordPress.
