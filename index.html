WP One Metric is the WordPress Plugin

This plugins visualizes the evaluation of your post content based on the score of One Content Metric to Rule Them All. This analyzes the latest 40 posts. After that, it analyzes once a day and automatically update the score.

Analyses are based on the following numbers as the score:

  • Unique page views by Google Analytics
  • Number of shares with Facebook
  • Number of shares with Twitter

How use

  1. Set up on Settings > WP One Metric Setting
  2. Push Analyze button on Dashboard WP One Metric

  2. Activate the plugin through the 'Plugins' menu in WordPress.
